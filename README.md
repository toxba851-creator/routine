# Routine — Veille Quotidienne

Agent de veille automatique pour Pape Malle BA (graphiste, expert IA, Dakar).

## Structure

```
reports/     → Rapports journaliers (YYYY-MM-DD.md)
sites.txt    → URLs à surveiller chaque matin
```

## Ajouter des sites à surveiller

Édite `sites.txt` et ajoute une URL par ligne :
```
https://monsite.com
https://marketplace-ia.com
```
